# Tic-Tac-Toe Game

This is a simple **Tic-Tac-Toe** game built using **Python** and **Tkinter**. The game allows two players to take turns and compete in a classic 3x3 Tic-Tac-Toe board.

---

## Features

- **Player vs Player**: Two players can play the game, with one using "X" and the other using "O".
- **Win Detection**: The game automatically checks if a player has won after every move.
- **Draw Detection**: If all moves are made and no one wins, the game declares it a tie.
- **User-Friendly UI**: A simple and clean interface with buttons for each cell.

---
